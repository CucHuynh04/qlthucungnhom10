# Sound Files

Để có âm thanh tùy chỉnh cho app, vui lòng thêm các file âm thanh sau vào thư mục này:

## Files cần thiết:

1. **click.mp3** - Âm thanh click nhẹ khi bấm button (giống Facebook)
   - Độ dài: ~100-200ms
   - Tần số: Nhẹ, thấp (~2000-4000 Hz)

2. **success.mp3** - Âm thanh khi thao tác thành công
   - Độ dài: ~300-500ms
   - Tần số: Cao, sáng (~3000-5000 Hz)

3. **error.mp3** - Âm thanh khi có lỗi
   - Độ dài: ~200-400ms
   - Tần số: Thấp, nghiêm túc (~1000-3000 Hz)

## Nguồn tìm kiếm file âm thanh miễn phí:

- [freesound.org](https://freesound.org/)
- [zapsplat.com](https://www.zapsplat.com/)
- [pixabay.com/sound-effects/](https://pixabay.com/sound-effects/)

## Lưu ý:

- Format: MP3 hoặc WAV
- Tối đa 500ms mỗi file
- Không quá lớn (< 100KB)
- Nếu không có file, app sẽ tự động dùng system sound mặc định
